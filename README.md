**DigitalLibrary** - веб-приложение для управления цифровой библиотекой.

_Помогает_:
* Учитывать прочитанные книги
* Искать книги по автору и жанру
* Оставлять заметки о прочитанном
* Оценивать книги
* Планировать, что прочитать следующим

* **HTML** — структура и разметка страницы
* **CSS** — стилизация и адаптивный дизайн
* **JavaScript** — интерактивность
* **Figma** — создание макета и дизайна
* **Live Server** — локальный сервер для разработки
* **Prettier** — автоматическое форматирование кода
